#malias7-14
Nuxt 3 によるモジュール構成を採用しています。
## 主な機能

- お気に入りの画像登録（localStorage対応）
- ズーム機能による画像の詳細表示
- 認証による公開・非公開フォルダの切り替え

## 使用技術

- Nuxt 3 / Vue.js
- Pinia / medium-zoom / VueMagnifier
- WordPress REST API

## 今後の展望

- ユーザー認証を強化し、パーソナライズされたフォルダ表示を実現する
- 画像キャッシュ機能により、表示速度の最適化
- Vue 3 の Composition API に全面移行してよりモジュール化
